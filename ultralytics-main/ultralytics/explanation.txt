1. __pycache__ : 存放 Python 编译后的字节码文件（ .pyc ），提高加载效率。
2. assets : 存放项目中使用的静态资源文件，例如图像、模型文件等。
3. cfg : 存放配置文件，通常用于模型参数和训练配置。
4. data : 数据相关的文件夹，包括数据集和数据处理脚本。
5. engine : 可能包含训练和推理引擎相关的代码，管理模型的训练、评估等流程。
6. hub : 存放模型和资源的目录，可能是模型仓库的实现。
7. models : 主要存放模型架构和定义文件。
8. nn : 包含神经网络层的定义或其他神经网络相关的代码。（一般魔改 就是从这里改起）
9. solutions : 可能存放解决方案或特定任务的代码，例如目标检测、分割等。
10. trackers : 存放跟踪算法或相关实现的代码。
11. utils : 工具类文件，通常包含辅助功能和公共函数。
12. __init__.py : Python 包的初始化文件，表示该目录是一个 Python 包。
